# Game Title: Space Shooter (Updated)

## Game Description

Space Shooter (Updated) is an action-packed space combat game that involves shooting down enemy spaceships. The player controls a futuristic fighter spaceship that destroys incoming enemy ships in deep space.

## How to Play

The objective is to kill all the enemy spaceships before getting destroyed by their attacks. You need to shoot down all enemy ships as soon as they arrive to progress through each level.

## Controls

* Move left: A / Left arrow
* Move right: D / Right arrow
* Shoot: Space bar
* Pause: P

## Features

* Neon-inspired space graphics
* Endless enemy wave generation
* Difficulty progression system (level scaling)
* Health-dependent enemy systems
* Particles exploding
* Damage-induced screen flashing
* Tracking system for scores and levels
* Sound effects with procedural sound synthesis
* Pause and resume system
* Canvas HTML5 gameplay

## Technology Used

* Engine: HTML5 Canvas
* Programming Language: JavaScript (ES6)
* Styling: CSS (Neon user interface design)
* Platform: Browser

## Instructions on Running

1. Clone/download project files
2. Launch `index.html` file using a browser
3. Click **Play Game** button and start playing
